# NEXUS Character Sheet — Owlbear Rodeo Extension

## Installation

1. Host this folder on your custom domain (e.g. `https://rodentroyaltywiki.com/nexus-obr-extension/`)
2. In Owlbear Rodeo, open **Extras → Extensions → Add**
3. Paste your manifest URL: `https://rodentroyaltywiki.com/nexus-obr-extension/manifest.json`
4. Enable the extension in your room

## Files

- `manifest.json` — OBR extension manifest (must be hosted at your domain)
- `index.html` — The full character sheet
- `icon.svg` — Extension icon shown in OBR toolbar

## Data Storage

Character data is saved to your **Owlbear Rodeo player metadata** so it persists across sessions and devices automatically. When OBR is not available (standalone use), localStorage is used as a fallback.
